1. CAUSA RAÍZ IDENTIFICADA
═══════════════════════════════════════════════════════════════════════════════

Línea 206 en Register():
	IsActive = userType == "Cliente" ? true : false

	→ Clientes: IsActive = true ✅
	→ Empleados: IsActive = false ❌

Línea 49 en Login():
	if (user == null || !user.IsActive) → RECHAZA LOGIN

	→ Si empleado tiene IsActive = false → RECHAZADO

RESULTADO: Empleados nunca pueden hacer login

5. INSTRUCCIONES DE INSTALACIÓN

PASO 1: Ejecutar Script SQL

   Archivo: Migrations/MigrationEmpleados.sql

   Comando SQL:

   SET [IsActive] = 1
   WHERE [TipoUsuario] = 'Empleado' AND [IsActive] = 0;
